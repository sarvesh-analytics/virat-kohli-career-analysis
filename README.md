# Virat Kohli Career Statistics Dashboard

## Project Overview

This project analyzes Virat Kohli's batting performance across different years using Excel and Power BI.

## Tools Used

* Microsoft Excel
* Power BI

## Key Insights

* Year-wise batting performance
* Strike rate trends
* Century analysis
* Boundary analysis
* Dot ball percentage trends

## Dashboard Features

* Interactive filters
* KPI cards
* Performance trends
* Comparative analysis

* Dataset.xlsx
* Dashboard.pbix
* Dashboard.pdf
